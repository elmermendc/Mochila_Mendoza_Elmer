Comandos básicos de Git:
git init: Inicializa un repositorio Git en el directorio actual.
git clone <URL>: Clona un repositorio remoto en tu máquina local.
git add <archivo>: Agrega un archivo al área de preparación.
git commit -m "mensaje": Crea un nuevo commit con los cambios en el área de preparación y un mensaje descriptivo.
git status: Muestra el estado actual del repositorio.
git log: Muestra el historial de commits.
git push: Envía los cambios locales al repositorio remoto.
git pull: Obtiene los últimos cambios del repositorio remoto y los fusiona con tu rama local.
git branch: Muestra las ramas existentes y resalta la rama actual.
git checkout <rama>: Cambia a la rama especificada.
git merge <rama>: Fusiona la rama especificada con la rama actual.
git remote add <nombre> <URL>: Asocia un repositorio remoto a tu repositorio local.
Comandos básicos de GitHub:

git remote -v: Muestra los repositorios remotos vinculados a tu repositorio local.
git push -u <nombre-remoto> <rama>: Envía los cambios locales a un repositorio remoto y establece la rama upstream (predeterminada).
git pull <nombre-remoto> <rama>: Obtiene los últimos cambios de un repositorio remoto y los fusiona con tu rama local.
git clone <URL>: Clona un repositorio remoto en tu máquina local.
git fork: Crea una copia personalizada de un repositorio remoto en tu cuenta de GitHub.
git pull-request: Crea una solicitud de extracción para enviar tus cambios a un repositorio original.
git merge <rama>: Fusiona la rama especificada con la rama actual.
git stash: Guarda temporalmente los cambios sin realizar un commit.
git revert <commit>: Deshace un commit específico y crea un nuevo commit con los cambios revertidos.
